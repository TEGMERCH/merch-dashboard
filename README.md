# Merch HQ — Sales Dashboard

A merchandise sales dashboard for tracking performance across multiple stores and POS systems.

## Features
- Upload Revel Product Mix CSV exports
- Auto-detects store name from filename
- Syncs live from Google Sheets
- Persists data in browser localStorage
- Date range filtering across all views
- Store rankings, comparisons, trends & product breakdowns

## Live Dashboard
Once GitHub Pages is enabled, your dashboard will be live at:
`https://TEGMERCH.github.io/merch-dashboard/`

## Data Sources
- **CSV Upload:** Export Product Mix reports from Revel and upload directly
- **Google Sheets Sync:** Click "Sync Sheet" to pull latest data from the connected Google Sheet
